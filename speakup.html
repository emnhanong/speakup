🔥 Kết nối Firebase Cloud
Nhập config để đồng bộ dữ liệu trên nhiều thiết bị. Không bắt buộc — ứng dụng chạy offline hoàn toàn.
🔥 Cấu hình Firebase
Dán config từ Firebase Console của bạn để kết nối database.
📋 Hướng dẫn nhanh (~3 phút):
  1. Vào console.firebase.google.com → Add project
  2. Authentication → Sign-in method → bật Email/Password
  3. Firestore Database → Create database → Production mode
  4. Project Settings ⚙️ → Add app (Web) → Copy firebaseConfig
